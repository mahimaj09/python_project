📚 Student Management System (Python)
📌 Project Overview

This project is a Student Management System developed using Python. It helps in managing student records efficiently by allowing users to add, update, delete, and view student details.

🎯 Objective
The main objective of this project is to:

Simplify student data management
Provide an easy-to-use interface for handling records
Demonstrate practical use of Python concepts like functions, file handling, and data structures

⚙️ Features
➕ Add new student details
📋 View all student records
✏️ Update existing student information
❌ Delete student records
🔍 Search student by ID or name

🛠️ Technologies Used
Python 
File Handling / Data Structures (Lists / Dictionaries)
